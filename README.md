# swfscGenetics

## Description

Miscellaneous functions for genetic data manipulation

## Installation

```r
# make sure you have Rtools installed
if (!require('devtools')) install.packages('devtools')

# install from GitHub
devtools::install_github('EricArcher/swfscGenetics')
```
